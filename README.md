## 项目简介
本项目是一个命令行工具，用于管理学生成绩数据。支持从 CSV 文件导入成绩，按学号/姓名查询，计算总分与平均分，按平均分排名，以及增删改查和结果导出功能。

## 核心功能列表
- [x] 从文件导入学生成绩数据（学号、姓名、多科成绩）
- [x] 按学号或姓名查询学生成绩
- [x] 计算每个学生的总分与平均分
- [x] 按平均分排序并输出排名
- [x] 新增、删除、修改学生成绩
- [x] 将统计结果导出为新文件
